# github-sdk

Desafio da Gazeus Games.

## Pré-requisitos

Ter as seguintes ferramentas instaladas.

#### rbenv

```sh
brew install rbenv
echo 'eval "$(rbenv init -)"' >> ~/.zshrc
```

#### Bunder

```sh
gem install bundler
```

#### Ruby:

```sh
rbenv install 3.1.6
```

## Execução do projeto

#### Gems:

```sh
bundle install
```
